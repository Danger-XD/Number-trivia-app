Trivia Generator App

Welcome to the Trivia Generator App!

How It Works:

  1. Generate Random Trivia: If no trivia type and no number is provided, clicking the "Generate" button will produce a random trivia fact from any category. If you provide a number and do not select a type, it will generate a random trivia of that number.
  2. Trivia Type Selection: If you select a trivia type but do not provide a number, the app will not generate any trivia. Ensure you provide a corresponding number or date for the selected trivia type.
  3. Date Type Trivia: When selecting the "Date" type, make sure to enter the value in the format of dd/mm. This format ensures the app understands and correctly processes the date you provide.
  4. Year Type Trivia: For generating trivia based on the "Year" type, enter the year in the format of yyyy. This format helps the app accurately retrieve facts related to the specified year.
